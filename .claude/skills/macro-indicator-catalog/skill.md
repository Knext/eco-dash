# Macro Indicator Catalog — 거시지표 카탈로그 작성 방법

## 워크플로우

### 1. 지표 분류
지표를 다음 6개 카테고리로 분류한다:
1. 변동성·공포 (VIX, MOVE, SKEW)
2. 인플레이션 (CPI, Core CPI, PPI, PCE, BEI)
3. 금리·유동성 (각종 채권금리, 스프레드, Fed Funds, M2)
4. 한국 매크로 (수출입, 무역수지, 환율, KOSPI EPS)
5. 글로벌 활동성 (DXY, 유가, 구리/금, BDI)
6. 신용 스트레스 (HY 스프레드, TED, 회사채 스프레드)

### 2. 지표별 메타데이터 표 작성

| 지표 | 카테고리 | 의미(1줄) | 정상 범위 | 경계 | 위험 | 선/동/후행 | 갱신주기 | 시그널 방향 |

**임계치 산정 규칙**:
- 가능하면 1990년 이후 분포의 백분위로 정의 (예: VIX 80th percentile ≈ 25)
- 학술/업계 컨센서스가 있는 경우 (예: 10Y-2Y < 0 = 침체 시그널) 우선 사용
- 임의의 라운드 넘버는 피하되, 직관성 위해 반올림은 허용

### 3. 핵심 조합 시그널 식별
2~3개 지표를 결합해서만 의미 있는 시그널을 별도 섹션으로 분리.

### 4. 한국 투자자 관점 보강
- 환율과의 상관: 모든 미국 시장 지표는 USD/KRW 영향을 함께 명시
- KOSPI 연동성: 한국 수출 데이터가 미국 매크로보다 KOSPI에 더 빠르게 반응
