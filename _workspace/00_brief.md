# 사용자 Brief (Phase 0)

작성일: 2026-05-16
프로젝트: 거시지표 기반 투자 대시보드

## 1. 투자 시간축
**중기 (수개월 ~ 1년)**
- 시그널 해석: 거시 구간 판단 중심, 시그널 노이즈 설정 보통
- 알림 주기: 일 단위로 충분, 실시간 불필요

## 2. 자산군 (다중)
- **한국 주식** (KOSPI/코스닥) — 한국 매크로 지표(수출입/환율/외국인 수급) 비중↑
- **미국 주식** (S&P500/나스닥) — VIX/금리/CPI 중심
- **글로벌 ETF/채권** — DXY, 장기채, 금/원자재 디자인도 포함

## 3. 알림 채널
**인앱(대시보드) 알림만** — 외부 채널 비활성, 메인 화면 상단 배너 + 알림 피드로 충분

## 4. 운영 형태
**혼자 사용 (로컬/개인 서버)**
- 인증 시스템 불필요
- SQLite + Vercel Hobby 또는 로컬 Node.js로 충분
- 비용 목표: 거의 0원

## 핵심 시사점 (팀 전체에 공유)

1. **3개 자산군 모두**를 의식한 지표 셋이 필요 (한국+미국+글로벌)
2. **알림은 인앱만** — 푸시/이메일 인프라 설계 생략 가능
3. **중기 시간축** — VIX/일일 변동성 알림보다 CPI/금리/스프레드/한국 수출 같은 월·주 단위 시그널 비중↑
4. **개인 사용** — 인증·다중 사용자 고려 불필요, SQLite로 단순화
5. **레짐 분류가 핵심** — "지금이 risk-on/off/recession 어디인가"가 매수매도 판단의 1차 신호
